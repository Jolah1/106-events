Questions people ask

What does it cost?

Nothing while we're in early access, and guests never pay — every event on 106 is free to attend. When paid plans arrive they'll be for organizers, and everyone already inside keeps what they have.

Do my guests need to install anything?

No. Guests get a link that opens in their browser, or they reply to the WhatsApp message. Their pass is a QR code on a web page — no app, no account, no password.

What happens if the venue's internet dies?

The door keeps working. The scanner keeps a copy of the list on the phone, judges each code locally, and syncs everything back when the signal returns — with the original scan times.

Is it only for weddings?

No — weddings are just the hardest case. Anything with a guest list runs the same way: conferences with sessions, launches, burials, birthdays. If it has parts and a door, it fits.
